# Manusman

Real-time consulting overlay powered by Kimi + Manus AI. Fullscreen transparent HUD for calls, meetings, and research.

## Setup

```bash
git clone https://github.com/sixandeight/cluely.git
cd cluely
npm install
cp .env.example .env   # fill in KIMI_API_KEY and MANUS_API_KEY
npm start
```

## Keybinds

| Key | Action |
|-----|--------|
| Ctrl+B | Toggle overlay |
| Ctrl+H | Screenshot |
| Ctrl+1 | Meeting brief |
| Ctrl+2 | Company snapshot |
| Ctrl+3 | Deal status |
| Ctrl+4 | Number lookup |
| Ctrl+5 | Who is this? (screenshot) |
| Ctrl+6 | Fact check (screenshot) |
| Ctrl+7 | Competitive intel (screenshot) |
